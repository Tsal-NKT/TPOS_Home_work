Для запуска Docker от имени пользователя, необходимо добавить этого пользователя в группу docker

Если в разделе REPOSITORY (docker images) нет дополнительных приписок кроме названия изображения, то это означает, что контейнер был скачан с DockerHub

После двоеточия в имени образа идет строковое обозначение версии контейнера

Проброс портов:
В команду docker run добавляется параметр -p <порт_хоста>:<порт_гостевой_ОС>

Просмотр логов контейнера:
Команда docker logs <имя_контейнера>. Параметр -f позволяет смотреть логи на запущенном контейнере в реалном времени

Просмотр подробных параметров контейнера:
Команда docker inspect <имя_контейнера>. Таким образом можно смотреть и параметры сети докера

Для подключения к БД MariaDB можно использовать не только ssh, но и специальный клиент
